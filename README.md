# NBC_W10-11-BookCart
`CoreData`와 `HTTP` 통신을 사용하여 책을 검색하고 저장하는 앱의 기능을 구현

## 프로젝트 구조
- `UIKit`
- `Rest API`
- `CoreData`
- `MVVM`
- `CodeBase`

## 필수 구현기능
- `Level 1` : `화면` 구성
- `Level 2` : `책 검색 화면` 구현
- `Level 3` : `책 상세 보기 & 담기 기능` 구현
- `Level 4` : `최근 본 책 기능` 구현

## 도전 구현기능
- `Level 5` : `무한 스크롤` 구현
